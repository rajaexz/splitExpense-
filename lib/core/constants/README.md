# Constants Directory

This directory contains all app-wide constants organized by type.

## Files

- **app_colors.dart**: All color constants with light/dark mode support
- **app_strings.dart**: All string constants for the app
- **app_fonts.dart**: Font family and size constants
- **app_dimensions.dart**: Padding, margin, radius, and size constants
- **app_routes.dart**: Route name constants

## Usage

Import constants as needed:

```dart
import 'package:jobcrak/core/constants/app_colors.dart';
import 'package:jobcrak/core/constants/app_strings.dart';
import 'package:jobcrak/core/constants/app_dimensions.dart';
```

## Adding New Constants

1. Add colors to `app_colors.dart`
2. Add strings to `app_strings.dart`
3. Add dimensions to `app_dimensions.dart`
4. Add routes to `app_routes.dart`

Keep constants organized and avoid duplication.
